### A typical HTTP Request ![HTTP Request](images/stack/http-request.png)
### Time to First Byte (TTFB) Duration of time from the moment the user makes a request to when the first byte arrives from the server.
This includes all Database Queries, logic, and API calls necessary to 'calculate' the document to be delivered to the client.

### Lets talk L(E/A)MP The most comment stack used in WordPress
### Linux The operating system the server is running
### Apache/Nginx The HTTP Server used to accpet requests via HTTP
Apache and Nginx can be run together. Nginx can also function as a reverse proxy between Apache and Nginx, or as a standalone proxy to PHP.
### MySQL/MariaDB The persistent data storage for the L(A/E)MP stack.
Via "drop-ins" WordPress can support PostgrSQL however this is not official support.
### PHP Currently supporting PHP 5.2+.
Looking to support 5.6+ in April 2019...
And PHP 7.0+ in December of 2019
![Rejoice](images/stack/rejoice.gif)
### So how can we use this? Fine tune our themes and plugins
Make as few DB calls as possible on initital load
Use AJAX to populate areas of the site that require large queries
Use transients to cache data like API calls
Deploy object and page caching where possible
Reduces 'decisions' and external resoruces to help improve TTFB
